# Welcome to Acceleratty

Start creating your documents here.

## Getting Started

1. Click **+ File** in the sidebar to create a new document
2. Write your content — this editor supports **Markdown** formatting
3. Click **Save** to save your work
4. Use the sync bar at the bottom to share changes with your team

## Markdown Quick Reference

| Syntax | Result |
|--------|--------|
| `# Heading` | Big heading |
| `**bold**` | **Bold text** |
| `*italic*` | *Italic text* |
| `- item` | Bullet list |
| `` `code` `` | `inline code` |
